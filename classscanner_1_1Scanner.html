Detailed Description

Main scanner class that orchestrates file scanning and analysis.

The Scanner class provides functionality to scan directories, calculate file hashes, check them against a malware database, and log results. It uses multiple components including Base for database operations, Hasher for hash calculation, and Logger for output.
